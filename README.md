# DMG in NodeJs

After downloaded the project open terminal in folder of the project

## Running Command
```
$ npm install
```
and 
```
$ npm start
```
## APIs

`Login` -> [post] `/auth/login`<br/>
`Create User` -> [post] `/api/user/create`<br/>

Values
* username, password<br/>

`Get Me` -> [get] `/api/user`<br/>
